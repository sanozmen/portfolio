Global Coronavirus Map (Updated Daily)

This interactive world map shows the total number of 2019 Novel Coronavirus disease (COVID-19) infections. Daily updated data is retrieved from this data source

There are 3 different layers to inform the users. First "Confirmed Cases" shows the total number of infected people at those locations. Second one "Deaths" indicates total number deaths caused by this disease. Lastly, the layer "Recover" denotes the number of people who got recovered.

How to use the map

Click on radio-buttons on top-right corner to switch between layers. In order to open popup windows with more details such as name of location, date of data etc. click on circles displayed on the map.
