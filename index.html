A compilation of my projects

Below are some of the web mapping projects completed by myself. Tech stack I mostly use for building those projects comprises HTML,CSS; desktop gis software like QGIS, ArcMAP; web mapping libraries,servers and spatial databases like leaflet.js, mapbox, carto, Geoserver & PostgGIS and graphic design software such as Adobe Illustrator and Photoshop.

A self-configured & deployed production ready backend (PostGIS & Geoserver) is up and running on elastic cloud service of AWS, for being able to create more dynamic projects.
